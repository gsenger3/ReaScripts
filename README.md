# Miscellaneous Reaper Scripts

A collection of miscellaneous scripts for the Reaper Digital Audio Workstation that were created to improve the Primo Studios workflow.

## Scripts

- `Primo-Headphones FX Toggle.lua` - Toggles my Headphones Monitor Master Track FX container on/off and updates a button state on the toolbar.
- `Primo-Monitors FX Toggle.lua` - Toggles my Studio Monitors Monitor Master Track FX container on/off and updates a button state on the toolbar. This script is the same script as Primo-Headphones FX Toggle.lua except it points to a different named FX container.
- `Primo-Prepare Mastering Session.lua` - Sets random colors for all tracks, aligns them sequentially, and creates regions for each track for mastering.
- `__startup.lua` - A startup script that triggers the FX Toggle scripts to synchronize toolbar states when Reaper opens.
